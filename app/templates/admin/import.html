Импорт продуктов

Импорт продуктов из JSON файлов с данными из Citilink и DNS.

Эта операция стандартизирует характеристики продуктов и добавит их в базу данных.

Информация о процессе миграции
Процесс миграции на унифицированную модель продуктов

Мы перешли с нескольких таблиц для разных компонентов на единую модель продуктов. Вот основные преимущества:

  • Гибкость - хранение любого типа компонента в одной таблице
  • Стандартизация - все характеристики в едином формате
  • Расширяемость - легко добавлять новые типы продуктов
  • Независимость от поставщиков - продукты из разных источников в одной таблице
Процесс импорта данных
  1. Извлечение данных из JSON файлов от поставщиков (Citilink, DNS)
  2. Определение типа продукта на основе категорий
  3. Стандартизация характеристик с использованием словарей соответствия
  4. Сохранение в базу данных в виде унифицированных продуктов
