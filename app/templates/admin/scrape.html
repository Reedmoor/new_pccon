DNS Parser

Для парсинга DNS используйте локальные BAT файлы.

Инструкция по запуску:

Для парсинга DNS используйте BAT файлы в папке local_parser/:

  • parse_category.bat - основной файл для парсинга категорий DNS
  • run_old_dns_parser.bat - запуск старого DNS парсера
  • run_visible_parser.bat - запуск парсера с видимым браузером
  • local_dns_parser.py - локальный Python скрипт парсера

Для быстрого парсинга и загрузки:

  • run_local_parse_and_upload.bat - парсинг + автозагрузка в базу
  • upload_to_docker.py - загрузка готовых данных в Docker

Совет: Запускайте parse_category.bat и выбирайте нужную категорию из списка.
