# Citi_parser

"main.py" — стартовый файл. Указана основная логика программы. Также тут выбирается, по какой категории будет парсить сайт.

"queries.py" — содержит всю информацию, которая нужна для запросов.

"request_handler.py" — содержит одну функцию с запросом.

"data_processors.py" — содержит функции, которые структурируют данные и записывают в JSON.
